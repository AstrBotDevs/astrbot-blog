[1Panel](https://1panel.io/) is an open-source server management panel designed to simplify server deployment and operations. It provides a user-friendly interface for installing, configuring, and managing applications and services. It supports multiple operating systems and offers a rich plugin ecosystem, allowing users to extend functionality based on their needs.

[BTPanel](https://bt.cn/) is a popular server management tool widely used on Linux and Windows servers. It provides a graphical interface for server administration, website deployment, database management, and related operations. With one-click installation for common software and runtime environments, BTPanel significantly reduces the operational burden of server maintenance.

AstrBot has now been officially listed by both 1Panel and BTPanel in their Docker app stores, allowing users to deploy AstrBot quickly and conveniently.

For more details, visit:

- [Deploy AstrBot with 1Panel in one click](https://docs.astrbot.app/deploy/astrbot/1panel.html)
- [Deploy AstrBot with BTPanel in one click](https://docs.astrbot.app/deploy/astrbot/btpanel.html)
