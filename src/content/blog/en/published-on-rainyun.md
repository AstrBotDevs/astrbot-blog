[Rainyun](https://www.rainyun.com/about), founded in 2018, is a domestic cloud computing service provider with independent intellectual property. Rainyun provides users with simple, affordable, and reliable cloud products, including cloud servers, bare-metal servers, virtual hosting, game cloud services, object storage, CDN, and other well-received cloud services.

AstrBot has now been officially listed by Rainyun in the Rainyun App Store. Users can deploy AstrBot in one click through Rainyun Cloud Apps and related deployment options.
